📱 PROJECT OVERVIEW

Development and implementation of a comprehensive Marshal Management System for Kuwait Motorsport Town (KMT). This enterprise-level web application provides complete digital transformation for marshal registration, event management, attendance tracking, and administrative oversight.

The system includes a modern responsive design, bilingual interface (Arabic/English), real-time notifications, and comprehensive reporting capabilities.
