📋 System Overview

What is the System?

A comprehensive system for managing marshals in Kuwait Motor Town (KMT) events, providing centralized management for all marshals, electronic attendance registration, event and calendar management, detailed reports and statistics, and instant notification system.

📱 User Guide - Admin

🔐 Login

  1. Open the website
  2. Click on Login
  3. Enter email and password
  4. Click Login
Default email: admin@kmt.kw

👥 Add New Marshal

  1. From the sidebar, select Marshal Management
  2. Click Add Marshal
  3. Fill in the required information:
    • Employee ID: KMT-XXX (example: KMT-101)
    • Full Name
    • Email
    • Phone Number
    • Civil ID
    • Date of Birth
    • Nationality
    • Marshal Type: Select one or more
  4. Click Add
Note: An email will be sent to the marshal with login credentials.

📸 Upload Marshal Picture

  1. From the marshal list, click on the desired marshal
  2. Click Edit
  3. Click on the camera icon
  4. Select the image from your device
  5. Click Save
Required Specifications:
  • Maximum size: 5 MB
  • Supported formats: JPG, PNG, GIF, WEBP

🎪 Create New Event

  1. From the sidebar, select Event Management
  2. Click Add Event
  3. Fill in all required information
  4. Upload event image (optional)
  5. Click Create Event
Result: An automatic notification will be sent to all suitable marshals.

✅ Manage Registration Requests

  1. Select Attendance Management from the menu
  2. Select the desired event
  3. A list of all registrants and their statuses will appear
  4. For each marshal, you can:
    • Click ✓ Approve to accept registration
    • Click ✗ Reject to reject registration
    • Add a note in the notes field
Status Description Action
🟡 Pending Awaiting approval Requires admin decision
🟢 Approved Approved Notification sent to marshal
🔴 Rejected Rejected Notification sent to marshal

📱 User Guide - Marshal

🆕 System Registration (First Time)

Important Note: The admin adds your account first.
  1. You will receive an email containing login credentials
  2. Open the website
  3. Click Login
  4. Enter the email and password sent
  5. It is recommended to change the password from settings

📝 Register for an Event

  1. From the home page or Events
  2. All upcoming events will appear
  3. Click on the desired event
  4. Read the details carefully
  5. If the event is suitable, click Register Now
  6. A confirmation message will appear
Notes:
  • You cannot register if the event is full
  • You cannot register if your type does not match the requirement

❌ Cancel Registration

  1. Go to Events
  2. Find the event you registered for
  3. Click on the event
  4. Click Cancel Registration
  5. Confirm cancellation
Alert: Cancellation is not possible after admin approval except by contacting the admin.
