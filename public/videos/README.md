# Video Placeholders

Place your video files here:

- `hero-bg.mp4` - Main hero section background video (racing/drift footage)
- `marshal-bg.mp4` - Login/Signup page background video (marshal with flags)

## Recommended Specifications:
- Format: MP4 (H.264)
- Resolution: 1920x1080 or higher
- Duration: 10-30 seconds (will loop)
- File size: < 10MB for better performance

## Where to get videos:
- Your own KMT event footage
- Stock footage from Pexels, Pixabay (search: "motorsport", "racing", "drift")
- Professional videographer from KMT events
