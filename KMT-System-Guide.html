📋 نظرة عامة على النظام

ما هو النظام؟

نظام شامل لإدارة المارشلات في فعاليات نادي الكويت للسيارات (KMT)، يوفر إدارة مركزية لجميع المارشلات، تسجيل الحضور والانصراف إلكترونياً، إدارة الفعاليات والتقويم، تقارير وإحصائيات تفصيلية، ونظام إشعارات فوري.

🎯 لماذا نحتاج هذا النظام؟

✨ المميزات الرئيسية

👨‍💼 للأدمن (الإدارة)

1. لوحة التحكم الرئيسية

  • عرض إحصائيات شاملة للنظام
  • عدد المارشلات النشطين
  • الفعاليات القادمة والحالية
  • نسبة الحضور العامة

2. إدارة المارشلات

  • ✅ إضافة مارشال جديد بكامل بياناته
  • ✅ تعديل بيانات المارشلات
  • ✅ رفع صور الملفات الشخصية
  • ✅ تحديد أنواع المارشال (Drag Race, Drift, Circuit)
  • ✅ تفعيل/إيقاف حسابات المارشلات
  • ✅ عرض سجل حضور كل مارشال

3. إدارة الفعاليات

  • ✅ إنشاء فعالية جديدة
  • ✅ تحديد نوع الفعالية والمتطلبات
  • ✅ تحديد الحد الأقصى للمارشلات
  • ✅ تحديد تاريخ البداية والنهاية
  • ✅ رفع صور الفعالية
  • ✅ إلغاء أو إتمام الفعاليات

4. إدارة الحضور

  • ✅ عرض قائمة المسجلين لكل فعالية
  • ✅ الموافقة أو رفض طلبات التسجيل
  • ✅ إضافة ملاحظات على الحضور
  • ✅ طباعة سجلات الحضور

5. التقارير والإحصائيات

  • 📊 تقرير حضور المارشلات
  • 📊 تقرير الفعاليات
  • 📊 تقرير الأداء الشهري
  • 📊 إحصائيات مفصلة بالرسوم البيانية
  • 🖨️ طباعة وتصدير التقارير

👨‍🔧 للمارشال

1. لوحة المعلومات الشخصية

  • عرض البيانات الشخصية
  • عرض نوع المارشال المسجل
  • عدد الفعاليات المشارك فيها
  • آخر الإشعارات

2. الملف الشخصي

  • ✅ عرض جميع البيانات الشخصية
  • ✅ تعديل البيانات (الاسم، الجوال، إلخ)
  • ✅ رفع صورة شخصية
  • ✅ تغيير كلمة المرور

3. الفعاليات

  • 📅 عرض جميع الفعاليات القادمة
  • 📅 عرض تفاصيل كل فعالية
  • ✅ التسجيل في الفعاليات المناسبة
  • ❌ إلغاء التسجيل قبل الموعد
  • 📋 عرض حالة التسجيل (معلق، مقبول، مرفوض)

4. سجل الحضور

  • 📊 عرض جميع الفعاليات السابقة
  • 📊 حالة الحضور لكل فعالية
  • 📊 ملاحظات الإدارة

5. الإشعارات

  • 🔔 إشعار بإضافة فعالية جديدة
  • 🔔 إشعار بقبول التسجيل
  • 🔔 إشعار بأي تغييرات في الفعالية
  • 🔔 تذكير قبل موعد الفعالية

📱 دليل الاستخدام - الأدمن

🔐 تسجيل الدخول

  1. افتح الموقع
  2. اضغط على تسجيل الدخول
  3. أدخل البريد الإلكتروني وكلمة المرور
  4. اضغط دخول
البريد الإلكتروني الافتراضي: admin@kmt.kw

👥 إضافة مارشال جديد

  1. من القائمة الجانبية، اختر إدارة المارشلات
  2. اضغط إضافة مارشال
  3. املأ البيانات المطلوبة:
    • رقم الموظف: KMT-XXX (مثال: KMT-101)
    • الاسم الكامل
    • البريد الإلكتروني
    • رقم الجوال
    • الرقم المدني
    • تاريخ الميلاد
    • الجنسية
    • نوع المارشال: اختر واحد أو أكثر
  4. اضغط إضافة
ملاحظة: سيتم إرسال بريد إلكتروني للمارشال ببيانات الدخول.

📸 رفع صورة لمارشال

  1. من قائمة المارشلات، اضغط على المارشال المطلوب
  2. اضغط تعديل
  3. اضغط على أيقونة الكاميرا
  4. اختر الصورة من جهازك
  5. اضغط حفظ
المواصفات المطلوبة:
  • الحجم الأقصى: 5 MB
  • الصيغ المدعومة: JPG, PNG, GIF, WEBP

🎪 إنشاء فعالية جديدة

  1. من القائمة الجانبية، اختر إدارة الفعاليات
  2. اضغط إضافة فعالية
  3. املأ جميع البيانات المطلوبة
  4. ارفع صورة للفعالية (اختياري)
  5. اضغط إنشاء الفعالية
نتيجة: سيتم إرسال إشعار تلقائي لجميع المارشلات المناسبين.

✅ إدارة طلبات التسجيل

  1. اختر إدارة الحضور من القائمة
  2. اختر الفعالية المطلوبة
  3. ستظهر قائمة بجميع المسجلين وحالاتهم
  4. لكل مارشال، يمكنك:
    • اضغط ✓ موافقة لقبول التسجيل
    • اضغط ✗ رفض لرفض التسجيل
    • أضف ملاحظة في خانة الملاحظات
الحالة الوصف الإجراء
🟡 معلق في انتظار الموافقة يتطلب قرار من الأدمن
🟢 مقبول تمت الموافقة تم إرسال إشعار للمارشال
🔴 مرفوض تم الرفض تم إرسال إشعار للمارشال

📱 دليل الاستخدام - المارشال

🆕 التسجيل في النظام (أول مرة)

ملاحظة مهمة: الأدمن هو من يضيف حسابك أولاً.
  1. ستستلم بريد إلكتروني يحتوي على بيانات الدخول
  2. افتح الموقع
  3. اضغط تسجيل الدخول
  4. أدخل البريد وكلمة المرور المرسلة
  5. يُنصح بتغيير كلمة المرور من الإعدادات

📝 التسجيل في فعالية

  1. من الصفحة الرئيسية أو الفعاليات
  2. ستظهر جميع الفعاليات القادمة
  3. اضغط على الفعالية المطلوبة
  4. اقرأ التفاصيل جيداً
  5. إذا كانت الفعالية مناسبة، اضغط سجل الآن
  6. ستظهر رسالة تأكيد
ملاحظات:
  • لن تتمكن من التسجيل إذا كانت الفعالية ممتلئة
  • لن تتمكن من التسجيل إذا كان نوعك لا يتطابق مع المطلوب

❌ إلغاء التسجيل

  1. اذهب إلى الفعاليات
  2. ابحث عن الفعالية التي سجلت فيها
  3. اضغط على الفعالية
  4. اضغط إلغاء التسجيل
  5. أكد الإلغاء
تنبيه: لا يمكن الإلغاء بعد موافقة الإدارة إلا بالتواصل مع الأدمن.

🛠️ التقنيات المستخدمة

البنية التقنية

الواجهة الأمامية (Frontend)

  • Next.js 16 - إطار عمل React متقدم
  • React 19 - مكتبة بناء الواجهات
  • Tailwind CSS - تصميم عصري وسريع
  • Framer Motion - حركات وانتقالات سلسة

الخادم (Backend)

  • Next.js API Routes - واجهات برمجية آمنة
  • NextAuth.js - نظام مصادقة متقدم
  • Prisma ORM - إدارة قاعدة البيانات

قاعدة البيانات

  • PostgreSQL - قاعدة بيانات موثوقة
  • Neon - استضافة سحابية آمنة
  • نسخ احتياطي تلقائي

التخزين السحابي

  • Cloudinary - تخزين الصور
  • 25 GB مساحة مجانية
  • ضغط تلقائي للصور
  • CDN عالمي سريع

الاستضافة

  • Vercel - استضافة عالمية
  • HTTPS آمن
  • تحديثات تلقائية
  • 99.9% وقت تشغيل

🔒 الأمان والخصوصية

إجراءات الأمان

الإجراء الأمني التفاصيل
🔐 تشفير البيانات جميع كلمات المرور مشفرة بخوارزمية bcrypt
🔒 HTTPS/SSL جميع الاتصالات مشفرة وآمنة
👤 صلاحيات الوصول كل مستخدم له صلاحيات محددة
💾 النسخ الاحتياطي نسخ احتياطي يومي تلقائي
🛡️ حماية من الاختراق حماية من SQL Injection و XSS
🔑 إدارة الجلسات انتهاء الجلسة التلقائي بعد فترة

الخصوصية

  • ✅ البيانات الشخصية سرية ومحمية
  • ✅ لا يتم مشاركة البيانات مع جهات خارجية
  • ✅ الصور محمية ولا يمكن الوصول لها إلا بصلاحية
  • ✅ لا يمكن للمارشال رؤية بيانات مارشلات آخرين
