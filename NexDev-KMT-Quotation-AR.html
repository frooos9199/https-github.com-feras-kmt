📱 نظرة عامة على المشروع

تطوير وتنفيذ نظام إدارة المارشالات المتكامل لنادي الكويت للسيارات (KMT). هذا التطبيق الإلكتروني على مستوى المؤسسات يوفر تحولاً رقمياً كاملاً لتسجيل المارشالات، إدارة الفعاليات، تتبع الحضور، والإشراف الإداري.

يتضمن النظام تصميماً عصرياً متجاوباً، واجهة ثنائية اللغة (عربي/إنجليزي)، إشعارات فورية، وإمكانيات تقارير شاملة.

⭐ الميزات المتضمنة

👥 بوابة المارشال

  • تسجيل المستخدم والمصادقة
  • نظام التسجيل في الفعاليات
  • إدارة الملف الشخصي
  • رفع رخصة القيادة (أمام وخلف)
  • سجل الحضور الشخصي
  • إشعارات فورية

👑 لوحة تحكم الإدارة

  • إدارة المارشالات (عرض/تعديل/إيقاف)
  • إنشاء وإدارة الفعاليات
  • مراقبة الحضور
  • قوائم حضور قابلة للطباعة
  • تقارير وإحصائيات شاملة
  • إدارة صور الرخص
  • نظام الإشعارات

🔒 الأمان والبنية التحتية

  • مصادقة آمنة (NextAuth.js)
  • كلمات مرور مشفرة (bcrypt)
  • قاعدة بيانات PostgreSQL (Neon)
  • تخزين صور سحابي (Cloudinary)
  • نسخ احتياطي تلقائي
  • تشفير SSL/HTTPS
  • ضمان توفر 99.9%

🌐 التقنيات المستخدمة

  • Next.js 16 (React 18)
  • TypeScript للموثوقية
  • Tailwind CSS تصميم متجاوب
  • Prisma ORM
  • استضافة Vercel (CDN عالمي)
  • ثنائي اللغة (عربي/إنجليزي)
  • واجهة متجاوبة للموبايل
